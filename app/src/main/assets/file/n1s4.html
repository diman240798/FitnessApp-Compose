Сгибание одной руки с рукояткой нижнего блока

Техника выполнения

Выжымания одной рукой

Выжымания обееми руками

Стоя лицом к тренажору. Рукоятку нижнего блока держать хватом снизу:
    - сделать вдох и согнуть руку;
    - по окончанию движения сделать выдох.

Это хорошое упражнение для изолированой "накачки" бицепсов.
источник: "Анатомия силовых упражнений." Фредерик Делавье
